# 🤖 WHAT IS SENTI-X?

SENTI-X is an AI-powered Sentiment Analysis System designed to understand, classify, and interpret human emotions from textual feedback.

The system automatically cleans text, extracts meaningful features, evaluates multiple machine learning models, selects the best performer, and predicts sentiment with confidence scores.

```text

         ▼

         ▼

         │
```
